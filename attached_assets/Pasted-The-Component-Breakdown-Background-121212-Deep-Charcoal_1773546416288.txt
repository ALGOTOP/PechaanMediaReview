Key Implementation Details:
Grid System: Uses display: grid with grid-template-columns: repeat(7, 1fr) to ensure the days line up perfectly regardless of screen size.

The "Available" Blocks: Those rounded gray squares use aspect-ratio: 1 to stay perfectly square and background: #27272a.

Selected Day: The white box uses color: black and font-weight: 700 to create that high-contrast look from your screenshot.

Responsiveness: The container is set to a fixed 380px width (standard for this widget), but you can change it to max-width: 100% to make it fluid for mobile.
